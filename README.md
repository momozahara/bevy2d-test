# BEVY2D-TEST

This is my first time learning [ECS](https://en.wikipedia.org/wiki/Entity_component_system).

## Control

**W A S D** - movement \
**E** - remove helmet (sprite on top of player) \
**F** - spawn npc on player location \
**G** - delete all of npc \
**[ ]** - zoom out/in
